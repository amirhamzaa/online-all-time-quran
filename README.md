# 📖 Online All Time Quran
**A seamless, ad-free, and high-quality Quranic audio streaming platform.**

---

## 🌟 Overview
**Online All Time Quran** allows you to listen to over 100+ world-renowned reciters and Islamic radio stations instantly without downloading any files. This project is built to provide a clean, distraction-free environment for spiritual growth.

## ✨ Key Features
* **🚫 100% Ad-Free:** Focus on your listening experience without any interruptions.
* **🎙️ 100+ Reciters:** Instant access to stations like Mishary Alafasi, Abdulbasit Abdulsamad, and more.
* **⚡ Zero Downloads:** Stream directly from high-quality sources, saving your device storage.
* **📱 Fully Responsive:** Beautifully optimized for Desktop, Tablet, and Mobile screens.
* **🔍 Smart Search:** Easily filter through stations by name, language, or category.
* **🌍 Multi-Language Support:** Includes Tafseer and translations in English, Urdu, French, Spanish, etc.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for a better user experience.
* **CSS3:** Modern UI with dark mode and custom animations.
* **JavaScript:** Dynamic station filtering and custom audio player logic.
* **FontAwesome:** For clean and intuitive iconography.
